menu microinteraction: szactrl.com

triggers :

  • mouse hover on wheel
  • mouse hover on menu items (music, merch, video, photo diary)

  • mouse click wheel

  • mouse hover off wheel
  • mouse hover off menu items (music, merch, video, photo diary)
  • mouse hover off entire menu microinteraction

rules :

  • mouse hover on wheel :
    - cursor change to pointer
    - wheel expands in size
  • mouse hover on menu items (music, merch, video, photo diary) :
    - cursor changes to pointer
    - color change to green

  • mouse click wheel :
    - tire moves right & rotates 360 degrees
    - menu items (music, merch, video, photo diary) fade in

  • mouse hover off wheel
    - wheel decreases to original size
  • mouse hover off menu items (music, merch, video, photo diary)
    - changes cursor to arrow
    - color change back to original color
  • mouse hover off entire menu microinteraction :
    - tire moves left & rotates 360 degrees in reverse
    - menu items (music, merch, video, photo diary) fade out
