triggers :

  • hover on menu items (music, merch, video, photo diary)
  • hover off menu items hover on menu items (music, merch, video, photo diary)
  • click menu items menu items (music, merch, video, photo diary)

  • hover on tire
  • hover off tire
  • click (toggle on) tire
  • click (toggle off) tire

rules

  • hover on menu items (music, merch, video, photo diary)
    - cursor changes to pointer
    - text color changes from gray to green
    - 5% increase in scale
  • hover off menu items (music, merch, video, photo diary)
    - cursor changes back to default
    - text color changes from green to gray
    - 5% decrease in scale (back to default)
  • click menu items (music, merch, video, photo diary)
    - 30% decrease in opacity
    - 5% decrease in scale (back to default)

  • hover on tire
    - cursor changes to pointer
  • hover off tire
    - cursor changes back to default

  • click (toggle on)tire
    - tire moves right 635 pixels, rotating 360 degrees
    - menu items (music, merch, video, photo diary) fade in one by one, starting from the left
  • click (toggle on)tire
    - tire moves left 635 pixels, rotating -360 degrees
    - menu items (music, merch, video, photo diary) fade out one by one, starting from the right
