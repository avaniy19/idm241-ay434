menu microinteraction: szactrl.com

triggers :

  • mouse hover on individual menu items
  • mouse hover off entire microinteraction

rules :

  • mouse hovering on each menu item changes cursor to pointer + "heartbeat" pulse animation with scale
  • increase scale → reset scale → increase scale → reset scale → pause
  • ↺ restart
  • mouse hovering off the entire microinteraction, tire moves right, and rotates 360° total. as mouse overlaps each menu item, menu item fades out to 0 opacity.
